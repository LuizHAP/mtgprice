  <files>src/bot/commands/start.ts</files>
  <action>
Create /start command handler:

1. Create src/bot/commands/start.ts
2. Import bot from src/lib/telegram.ts
3. Import comparePassword from src/lib/auth.ts (reuse auth utilities)
4. Import db client to update user

5. Implement /start command:
   - Read command arguments: ctx.match (contains password after /start)
   - Extract password from arguments
   - Compare password with process.env.BOT_PASSWORD using comparePassword
   - If password wrong:
     - Reply: "Invalid password. Access denied."
     - Return (don't link account)
   - If password correct:
     - Get chat ID from ctx.chat?.id
     - Update user in database: set telegramChatId = chatId.toString()
     - Reply: "Welcome! Your Telegram account is now linked. You'll receive notifications here."
     - Log successful linking

6. Register command on bot instance: bot.command('start', startCommandHandler)

Use exact pattern from RESEARCH.md lines 318-334 for reference.
