<task type="auto" tdd="true">
  <name>Task 3: Implement wishlist database queries</name>
  <files>src/lib/wishlist/queries.ts</files>
  <behavior>
    - getUserWishlist(userId) joins wishlists + cards tables, orders by addedAt DESC
    - addCardToWishlist(userId, cardId) inserts into wishlists table, handles duplicate (23505) error
    - removeCardFromWishlist(userId, cardId) deletes from wishlists table, returns rows affected
    - getLatestPricesForCard(cardId) queries prices table, filters by cardId, orders by timestamp DESC LIMIT 1 per source
    - getBestPrice(prices) finds minimum price across 4 sources
    - calculatePriceTrend(currentPrice, pricesHistory) compares current vs 7 days ago, returns trend object
  </behavior>
  <action>
    Create src/lib/wishlist/queries.ts with database query functions for wishlist management.

    Query functions to implement:

    1. **getUserWishlist(userId: number)** - Fetch user's wishlist with card metadata
       - Query: `SELECT { cards.*, wishlists.addedAt } FROM wishlists JOIN cards ON wishlists.cardId = cards.oracleId WHERE wishlists.userId = $userId ORDER BY wishlists.addedAt DESC`
       - Use Drizzle ORM: `db.select({...cards, addedAt: wishlists.addedAt}).from(wishlists).innerJoin(cards, eq(wishlists.cardId, cards.oracleId)).where(eq(wishlists.userId, userId)).orderBy(desc(wishlists.addedAt))`
       - Returns: Promise<WishlistItem[]>

    2. **addCardToWishlist(userId: number, cardId: string)** - Add card to wishlist
       - Query: `INSERT INTO wishlists (userId, cardId) VALUES ($userId, $cardId)`
       - Use Drizzle ORM: `db.insert(wishlists).values({ userId, cardId })`
       - Handle duplicate error (PostgreSQL error code 23505): throw new Error('Card already in wishlist')
       - Returns: Promise<void>

    3. **removeCardFromWishlist(userId: number, cardId: string)** - Remove card from wishlist
       - Query: `DELETE FROM wishlists WHERE userId = $userId AND cardId = $cardId`
       - Use Drizzle ORM: `db.delete(wishlists).where(and(eq(wishlists.userId, userId), eq(wishlists.cardId, cardId)))`
       - Returns: Promise<void>

    4. **getLatestPricesForCard(cardId: string)** - Get latest prices from all 4 sources
       - Query prices table for each source (liga_magic, tcgplayer, cardmarket, cardkingdom)
       - For each source: `SELECT * FROM prices WHERE cardId = $cardId AND source = $source ORDER BY timestamp DESC LIMIT 1`
       - Use Drizzle ORM with `and(eq(prices.cardId, cardId), eq(prices.source, source))`
       - Returns: Promise<{ ligaMagic: number | null, tcgplayer: number | null, cardmarket: number | null, cardkingdom: number | null }>

    5. **getBestPrice(prices: object)** - Find lowest price (pure function)
       - Extract all non-null price values
       - Return minimum price and its source
       - Returns: { source: string, priceBrl: number } | null

    6. **calculatePriceTrend(currentPrice: number, priceHistory: Price[])** - Calculate % change vs 7 days ago
       - Find price from 7 days ago (±6 hours tolerance)
       - Calculate: ((current - old) / old) * 100
       - Return: { trend: 'up' | 'down' | 'stable', percentChange: number | null }
       - If no price 7 days ago, return { trend: 'stable', percentChange: null }

    Import database client from src/db/index.ts, schema from src/db/schema/.

    File should be ~120 lines, export 6 functions, use Drizzle ORM queries.

    Reference pattern from Phase 2: src/db/queries/prices.ts
